# Machine Learning Techniques Practice 

This repository contains my hands-on practice of important Machine Learning techniques implemented using Python in Google Colab.

The goal of this repository is to explore ensemble learning, hyperparameter tuning, and dimensionality reduction using popular ML algorithms.

---

## Notebooks in this Repository

- Ensemble Learning.ipynb
- GridSearchCV.ipynb
- RandomSearchCV.ipynb
- PCADIMENSION.ipynb

Each notebook demonstrates a different Machine Learning concept with practical examples.
